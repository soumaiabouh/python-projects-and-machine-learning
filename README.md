# Python Projects
This repository contains a collection of Python projects, including a GUI calculator and several programs related to machine learning. These projects showcase a variety of Python programming techniques and demonstrate proficiency in using popular libraries such as NumPy and TensorFlow.

The GUI calculator project demonstrates proficiency in creating interactive graphical user interfaces using the Tkinter library. The machine learning programs include implementations of various algorithms such as linear regression, logistic regression, and neural networks. These programs are designed to showcase practical applications of machine learning techniques in areas such as image recognition and natural language processing.

This repository is intended to serve as a showcase of programming skills and to provide examples of how Python can be used to solve a wide range of problems.
