# Machine Learning Projects

This directory contains a collection of machine learning projects completed as part of an Introduction to Machine Learning course. Each subfolder represents a distinct project focusing on different machine learning algorithms and techniques.

## Project Overview

### 1. KNN and Decision Tree

**Location:** `KNN-and-Decision-Tree/`

This project involves implementing and comparing K-Nearest Neighbors (KNN) and Decision Tree algorithms. The implementation includes:

- **Algorithms:** KNN (K-Nearest Neighbors) and Decision Tree classifiers
- **Datasets:** 
  - Hepatitis dataset
  - Diabetic Retinopathy Debrecen (Messidor features) dataset
- **Languages & Tools:** Python
- **Key Activities:**
  - Algorithm implementation and evaluation
  - Performance comparison across two datasets
  - Data analysis and interpretation

**Files:**
- `implementation-code.ipynb` - Jupyter notebook containing the code implementation
- `hepatitis.csv` - Hepatitis dataset
- `messidor_features.csv` - Diabetic Retinopathy Debrecen dataset
- `README.md` - Project documentation

---

### 2. Logistic Regression & Multiclass Regression

**Location:** `Logistic-Regression-&-Multiclass-Regression/`

This project focuses on implementing logistic regression for sentiment analysis and multiclass regression for document classification. The project includes:

- **Algorithms:** Logistic Regression and Multiclass Regression
- **Applications:**
  - Sentiment analysis using the IMDB database
  - Document classification using the 20 Newsgroups database
- **Languages & Tools:** Python with scikit-learn, pandas, and numpy
- **Key Activities:**
  - Data preprocessing and feature engineering
  - Model training and evaluation
  - Performance analysis and comparison

**Files:**
- `implementation-code.ipynb` - Jupyter notebook containing the code implementation
- `README.md` - Project documentation

**Datasets:**
- IMDB Sentiment Analysis (external: http://ai.stanford.edu/~amaas/data/sentiment/)
- 20 Newsgroups Database (external)

---

### 3. MLP and ConvNet

**Location:** `MLP-and-ConvNet/`

This project involves comparing the performance of Multilayer Perceptron (MLP) and Convolutional Neural Network (ConvNet) algorithms for image classification. The project includes:

- **Algorithms:** 
  - Multilayer Perceptron (MLP)
  - Convolutional Neural Network (ConvNet)
- **Dataset:** Fashion-MNIST dataset
- **Languages & Tools:** Python with TensorFlow
- **Key Activities:**
  - Algorithm implementation using deep learning framework
  - Model training and testing
  - Performance comparison and analysis
  - Image classification accuracy evaluation

**Files:**
- `implementation-code.ipynb` - Jupyter notebook containing the code implementation
- `README.md` - Project documentation
